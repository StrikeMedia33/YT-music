# React Components

This directory contains reusable React components for the frontend.

## Component Categories

### Layout Components
- `Header.tsx` - Navigation header
- `Sidebar.tsx` - Navigation sidebar
- `Footer.tsx` - Page footer

### UI Components
- `Button.tsx` - Reusable button with variants
- `Table.tsx` - Data table with sorting/filtering
- `StatusBadge.tsx` - Status indicator badges
- `LoadingSpinner.tsx` - Loading indicator
- `ErrorMessage.tsx` - Error display component

### Form Components
- `FormInput.tsx` - Text input field
- `FormSelect.tsx` - Dropdown select
- `FormTextarea.tsx` - Multi-line text input

### Domain Components
- `ChannelCard.tsx` - Channel display card
- `VideoJobCard.tsx` - Video job display card
- `AudioTrackList.tsx` - List of audio tracks
- `ImageGallery.tsx` - Visual gallery display

## Guidelines

Components should:
- Be fully typed with TypeScript
- Use Tailwind CSS for styling
- Support dark mode where applicable
- Include loading and error states
- Be accessible (WCAG 2.1 AA)
