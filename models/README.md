# Database Models

This directory contains SQLAlchemy ORM models for database tables.

## Models

- `channels.py` - YouTube channel model
- `video_jobs.py` - Video generation job model
- `audio_tracks.py` - Generated audio track model
- `images.py` - Generated image/visual model
- `render_tasks.py` - FFmpeg render task model

## Database Session

- `database.py` - Database connection and session management
- `base.py` - Base class for all models
