# Utility Functions

This directory contains helper functions and shared utilities.

## Common Utilities

- `file_helpers.py` - File system operations (saving, loading, path management)
- `date_helpers.py` - Date/time formatting and parsing
- `validation.py` - Input validation helpers
- `logging_config.py` - Centralized logging configuration
- `config.py` - Configuration management and environment variable loading

## Guidelines

Utilities should:
- Be pure functions when possible
- Have clear, single responsibilities
- Include comprehensive error handling
- Be well-documented with examples
