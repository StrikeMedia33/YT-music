# API Routes

This directory contains FastAPI route handlers and endpoints for the AI Background Channel Studio.

## Structure

- Channel management routes (CRUD operations)
- Video job management routes
- Pipeline control endpoints (regenerate, rerender)
- Health check and monitoring endpoints

```python
# /api/channels.py - Channel management
# /api/video_jobs.py - Video job operations
# /api/health.py - Health check endpoint
```
