# Pydantic Schemas

This directory contains Pydantic models for API request/response validation.

## Schema Types

- **Request Schemas**: Validate incoming API requests (e.g., `ChannelCreate`, `VideoJobCreate`)
- **Response Schemas**: Format API responses (e.g., `ChannelResponse`, `VideoJobDetail`)
- **Update Schemas**: Handle partial updates (e.g., `ChannelUpdate`, `VideoJobUpdate`)

## Examples

```python
# /schemas/channels.py - Channel schemas
# /schemas/video_jobs.py - Video job schemas
# /schemas/audio_tracks.py - Audio track schemas
```
